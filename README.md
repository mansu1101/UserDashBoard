# UserDashBoard

Node.js scalable server for UserDashBoard application.

Node.js version - 8.x
Mongodb version - 3.6

Quick setup guide -
#In Script directory

1. bash initApp.sh


# In root directory

1. Install node.js v8.11.2
2. npm i -g typescript
3. npm i
4. bash scripts/build.sh
5. node index.js
